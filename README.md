# 🎨 Creative Design

## 📌 About

**Creative Design** is a simple static website built with **HTML5** and **CSS3**. The project presents a fictional web design company and was created to practice front-end development and website layout.

## ✨ Features

* Modern landing page layout
* Navigation bar
* "About Us" section with parallax-style background images
* Responsive header with background overlay
* Footer with social media icon
* Clean and minimal design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Raleway)
* Font Awesome

## 📂 Project Structure

```text
📁 Creative-Design/
├── index.html
├── style.css
└── img/
    ├── planet.jpg
    ├── coffee.jpg
    ├── mobile-phone.jpg
    └── offic.jpg
```

## 🚀 Getting Started

1. Clone or download this repository.
2. Make sure the **img** folder contains all required images.
3. Open `index.html` in your web browser.

## 🎯 Purpose

This project was created for learning purposes to improve HTML and CSS skills, including page structure, styling, and basic web design principles.
